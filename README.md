📝 Smart To-Do App
A simple and interactive To-Do List application built with Python and Streamlit. The app helps users efficiently manage daily tasks by providing an intuitive interface to add, organize, and filter tasks based on their status.
✨ Features
➕ Add new tasks
📋 View all tasks
🔍 Filter tasks (e.g., All, Completed, Pending)
✅ Mark tasks as completed
🗑️ Delete tasks
⚡ Simple and responsive interface
🌐 Built entirely with Python and Streamlit
🛠️ Tech Stack
Python
Streamlit
🚀 Getting Started
pip install -r requirements.txt
streamlit run app.py
📌 Purpose
This project demonstrates how to build an interactive task management web application using Python and Streamlit, with features for task creation, organization, filtering, and status management.
